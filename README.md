# Memory-Cores
## Overview
This is an assortment of cores related to memory.
If there is a memory core that seems to be missing from another project it is probably here.
|Folder |Contents                   |
|-------|---------------------------|
|MPMC11 |Multiport memory controller|
|MMU    |MMU related cores          |
|       |TLB     |                  |
|       |PTW     | page table walker|
|verilog|internal| cores using FPGA internal memory resources|
|       |external| external memory controllers -SRAM,PSRAM,DRAM|
|       |miscellaneous cores used by other projects|
